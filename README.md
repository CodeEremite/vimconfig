# 本vim配置使用方法
## linux用户
输入命令:
    git clone https://github.com/CodeEremite/vimconfig.git ~/.vim
    ln -s ~/.vim/.vimrc ~/.vimrc
备注：
1.这里默认已装好ctags,安装目录为: /usr/bin   如你的安装不是这样，请更改.vimrc中taglist设置
